# ROYA LLEATHER - Premium Leather Shoes Website

A luxury dark-themed website for ROYA LLEATHER, featuring premium custom leather shoes made in Pakistan.

## Tech Stack

- **Next.js 14** - React framework
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type safety
- **Lucide React** - Beautiful icons

## Features

- Dark luxury theme with gold accents
- Responsive design
- Premium typography (Playfair Display + Poppins)
- Radial gradient backgrounds
- Gold gradient buttons with shadows
- Product showcase with bestsellers
- WhatsApp ordering section

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Design Specifications

- **Background**: Deep black (#0B0B0B) with radial gradient
- **Accent Color**: Gold (#C9A24D, #D4AF37)
- **Fonts**: Playfair Display (headings), Poppins (body)
- **Layout**: 2-column hero, 3-column features, 4-column products, 3-column steps
